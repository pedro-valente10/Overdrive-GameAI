# Overdrive-GameAI 🏎️🎮

Um jogo de corrida top-down, estilo pixel-art retrô, desenvolvido com GODOT.

O objetivo é implementar na prática um sistema de NPCs que "pareçam" inteligentes utilizando path-finding, path smoothing e outros algorítimos se possíveis.
